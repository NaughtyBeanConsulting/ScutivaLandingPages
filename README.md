# Scutiva Landing Pages

Marketing site source for Scutiva OSS, built with Hugo for deployment to GitHub Pages.

This site uses custom layouts in `layouts/` and static assets in `static/`. It does not depend on a Hugo theme module.

## Local development

If Hugo is installed locally:

```bash
hugo server
```

## Deployment

Deployment is handled by GitHub Actions through `.github/workflows/deploy.yml` and publishes the generated Hugo site to GitHub Pages.

## Brand assets

This repo uses the Scutiva project logo and the Naughty Bean Coffee support badge copied from the main Scutiva repository.
